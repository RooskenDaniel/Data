Als dataset heb ik gekozen voor informatie over gekentekende voertuigen van de RDW.
De tabelen die ik heb gekozen zijn:
Open_Data_RDW__Gekentekende_voertuigen
Open_Data_RDW__Gekentekende_voertuigen_brandstof
Open_Data_RDW__Gekentekende_voertuigen_assen

Deze sets zijn behoorlijk groot, daarom heb ik ze niet in de github gedaan.
Als je wil kan je ze downloaden van de volgende links:
https://opendata.rdw.nl/Voertuigen/Open-Data-RDW-Gekentekende_voertuigen/m9d7-ebf2
https://opendata.rdw.nl/Voertuigen/Open-Data-RDW-Gekentekende_voertuigen_brandstof/8ys7-d773
https://opendata.rdw.nl/Voertuigen/Open-Data-RDW-Gekentekende_voertuigen_assen/3huj-srit

Je drukt hier op 'Exporteer' en dan 'CSV' (niet op 'CSV voor Excel').

In deze GitHub heb ik wel een kleinere versie van deze drie bestanden staan. Deze staan onder deel 2 en dan in het mapje data.
Ik heb gekozen om de bestanden in te korten omdat GitHub dit leuker vindt en omdat testen dan sneller gaat.
Zo hoeft er niet telkens in miljoenen voertuigen gezocht te worden, maar duizenden.
De API werkt wel met de originele bestanden. Als je dit wilt testen, vervang de bestanden in het mapje data dan.
Maar voor testdoeleinden lijkt de ingekorte versie van het bestand mij voldoende.
